# Plan 3: Git configuration — .gitconfig, .gitignore_global

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Create the git configuration files, add symlink (`link()`) calls to both install scripts, and create a per-plan smoke test script wired into CI. After this plan, a fresh machine running `install-macos.sh` or `install-wsl.sh` will have git configured with delta paging, difftastic, VS Code merge tool, conventional commit aliases, and a comprehensive global gitignore.

**Architecture:** Two config files in `git/`, modifications to both install scripts (adding `link()` calls), a new smoke test in `scripts/`, and a CI workflow update.

**Tech Stack:** git config (INI format), gitignore

**Design references:**
- `docs/design/git.md` — .gitconfig, .gitignore_global, conventional commits, delta config
- `docs/design/DESIGN.md` — repo structure, implementation order (step 4: git/)

**Design amendments (supersede git.md for this plan):**
- **User identity via `[include]`:** The `[user]` section is NOT committed. Instead, `.gitconfig` includes `~/.gitconfig.local` (gitignored) which contains per-machine `[user]` name/email. This avoids committing identity to a public repo.
- **`.gitconfig.local` added to `.gitignore` and `.gitignore_global`:** Alongside the existing `.bashrc.local` pattern in both files.
- **`.gitignore_global` expanded for financial institution patterns:** Added `*.env`, SSH private key patterns (`*_rsa`, `*_ecdsa`, `*_ed25519`), Java keystores (`*.jks`), Terraform state/vars (`terraform.tfstate*`, `.terraform/`, `*.tfvars`), npm/PyPI auth (`.npmrc`), Apple/Stripe keys (`*.p8`).
- **`unwip` alias improved:** Added feedback message when last commit is not a WIP.
- **Mergetool variables quoted:** `"$REMOTE" "$LOCAL" "$BASE" "$MERGED"` to prevent word-splitting on paths with spaces.
- **`navigate = true` documented inline:** Comment in .gitconfig explaining disabled less search.
- **Git-native alias prefix exception:** `undo`/`wip`/`unwip`/`aliases` are invoked as `git <alias>`, where `git` itself is the prefix. These do not follow the `g*` bash alias prefix because they are not bash aliases.
- **`cog`/`git-cliff` aliases:** Already covered by `cz`/`clog`/`cver` in `.bash_aliases` (Plan 2). Not duplicated in `.gitconfig`.

**Out of scope for this plan:**
- lazygit configuration (DESIGN.md step 8 — separate plan)
- gitleaks / prek pre-commit hooks (Plan 12 — prek)

**Prerequisites:**
- Plan 1 (Foundation) and Plan 2 (Bash) are merged to `tool-verification-script`
- Both install scripts have `link()` helper and existing bash link() calls from Plan 2

**Branch strategy:** New worktree on branch `plan3-git` off `tool-verification-script`. Push to remote, PR against `tool-verification-script`, merge when CI is green and hardware-tested.

```ini
# .gitconfig — git configuration
# User identity is in ~/.gitconfig.local (gitignored, per-machine).
# See docs/design/git.md for the specification.

[include]
    path = ~/.gitconfig.local

[core]
    pager        = delta
    excludesFile = ~/.gitignore_global
    autocrlf     = input

[interactive]
    diffFilter = delta --color-only

[delta]
    navigate        = true    # remaps n/N to jump between hunks; disables less search (/)
    side-by-side    = true
    line-numbers    = true
    syntax-theme    = Monokai Extended
    features        = decorations

[delta "decorations"]
    file-style                   = bold yellow
    file-decoration-style        = none
    hunk-header-style            = line-number syntax

[diff]
    colorMoved = default
    tool       = difftastic

[difftool]
    prompt = false

[difftool "difftastic"]
    cmd = difft "$LOCAL" "$REMOTE"

[merge]
    conflictstyle = diff3
    tool          = vscode

[mergetool "vscode"]
    cmd = code --wait --merge "$REMOTE" "$LOCAL" "$BASE" "$MERGED"

[mergetool]
    keepBackup = false

[pull]
    rebase = true

[rebase]
    autoStash   = true
    autoSquash  = true

[push]
    default        = current
    autoSetupRemote = true

[fetch]
    prune = true

[branch]
    sort = -committerdate

[log]
    date = relative

[stash]
    showPatch = true

[alias]
    undo    = reset HEAD~1 --mixed
    wip     = !git add -A && git commit -m "WIP"
    unwip   = !git log -1 --format='%s' | grep -q 'WIP' && git reset HEAD~1 || echo 'No WIP commit found — nothing undone'
    aliases = config --get-regexp alias
```

# Exit: 0 if all tests pass, 1 if any fail

pass=0
fail=0

ok() {
  printf "  \033[0;32m✓\033[0m %s\n" "$1"
  pass=$((pass + 1))
}

nok() {
  printf "  \033[0;31m✗\033[0m %s\n" "$1"
  fail=$((fail + 1))
}

check() {
  local desc="$1"; shift
  if "$@" >/dev/null 2>&1; then
    ok "$desc"
  else
    nok "$desc"
  fi
}

check "difftool difft cmd"           grep -q 'cmd.*=.*difft' "$REPO_ROOT/git/.gitconfig"

# Merge
check "merge.conflictstyle = diff3"  grep -q 'conflictstyle.*=.*diff3' "$REPO_ROOT/git/.gitconfig"
check "merge.tool = vscode"          grep -q 'tool.*=.*vscode' "$REPO_ROOT/git/.gitconfig"
check "mergetool vscode cmd"         grep -q 'code --wait --merge' "$REPO_ROOT/git/.gitconfig"
check "mergetool.keepBackup = false" grep -q 'keepBackup.*=.*false' "$REPO_ROOT/git/.gitconfig"

# Workflow settings
check "pull.rebase = true"          grep -q 'rebase.*=.*true' "$REPO_ROOT/git/.gitconfig"
check "rebase.autoStash = true"     grep -q 'autoStash.*=.*true' "$REPO_ROOT/git/.gitconfig"
check "rebase.autoSquash = true"    grep -q 'autoSquash.*=.*true' "$REPO_ROOT/git/.gitconfig"
check "push.default = current"      grep -q 'default.*=.*current' "$REPO_ROOT/git/.gitconfig"
check "push.autoSetupRemote = true" grep -q 'autoSetupRemote.*=.*true' "$REPO_ROOT/git/.gitconfig"
check "fetch.prune = true"          grep -q 'prune.*=.*true' "$REPO_ROOT/git/.gitconfig"
check "branch.sort"                 grep -q 'sort.*=.*-committerdate' "$REPO_ROOT/git/.gitconfig"
check "log.date = relative"         grep -q 'date.*=.*relative' "$REPO_ROOT/git/.gitconfig"
check "stash.showPatch = true"      grep -q 'showPatch.*=.*true' "$REPO_ROOT/git/.gitconfig"

# Aliases
check "alias: undo"                 grep -q 'undo.*=.*reset HEAD~1' "$REPO_ROOT/git/.gitconfig"
check "alias: wip"                  grep -q 'wip.*=.*!git add -A' "$REPO_ROOT/git/.gitconfig"
